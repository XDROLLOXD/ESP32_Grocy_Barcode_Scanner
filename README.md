# ESP32 Grocy Barcodescanner
Simple ESP32 Barcodescanner for Grocy with GM60 scanner

## Setup
   1. create a Grocy API-KEY
   2. Edit the Settings
   3. Wire everytihing
   4. Upload the .ino File with the Arduino IDE
